# cinema-website-index-record
该项目主要对电影与电视剧相关网站进行信息索引整理，通过统一格式记录站点的基础属性。仓库内容强调信息清晰度与可维护性，适合用于影视站点生态研究与数据参考。项目不包含任何视频文件或播放接口，仅整理站点层面的公开信息。所有资料均来自公开网络环境，保持客观中立立场。
